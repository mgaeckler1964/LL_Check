# LL_Check
 Livelink and web server analysis

You will need Borland C++ Builder to compile

Actually I'm using http_get to monitor my webserver "https:\\www.gaeckler.at\". My crontab starts http_get every hour, logs the execution time and reports errors in an mbox file.
